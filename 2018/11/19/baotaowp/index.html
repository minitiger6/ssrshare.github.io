发表于 2018-11-19 | 更新于 2019-02-17 | 分类于 博客搭建教程 | 评论数： | 阅读次数：

宝塔面板一键搭建WordPress，真的很方便

宝塔介绍

宝塔软件是一款linux/windows平台均可使用的服务器管理软件，自带环境包；由尊云公司投资收购的一个公司开发运营，主要基于centos操作系统，可一键包装nginx、apache、php、mysql、pureftpd、phpmyadmin等环境，自带在线网页管理面板，可用于建站、开通FTP、防火墙管理、以及强大的在线文件管理功

安装宝塔面板

建议选择centos7，官方也是这么建议的！

官网：宝塔面板一键搭建WordPress
一键安装代码：yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && bash install.sh

升级代码： curl http://download.bt.cn/install/update6.sh|bash

安装过程

  • 中间会出现一次确认，输入 y 即可
  • 安装完成后，会出现 ip+8888+神秘代码，账号，密码

我们复制ip那行，到浏览器里输入

登录成功后，出现推荐安装套件窗口，根据自己的选择部署即可

一般选择 LNMP 其性能更强

选择软件管理，一键部署源码

选择wordpress

填写域名就可以了，其他自动生成

进行数据库的配置

将之前生成数据库名和密码复制过来
