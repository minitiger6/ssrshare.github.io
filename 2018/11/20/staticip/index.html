谷歌云静态IP问题——用完了一定要释放Compute Engine Static Ip Charge

发表于 2018-11-20 | 更新于 2019-02-18 | 分类于 服务器脚本 | 评论数： | 阅读次数：

对于谷歌云的静态Ip问题，我查阅了很多的内容，可以没有得到我想要的东西。经过了自己的多次研究，得出了以下的结论。

如果建VM实例时选择固定Ip

  • 如果是对于代理服务器的话，选用静态Ip可能没有什么大用，因为谷歌云会给你提供一个动态的ip地址，一般我们的代理服务器不会出现什么状况，IP地址不会出现改变的，因此选用动态的就好。

删除VM实例后一定要释放ip

  • Compute Engine Static Ip 是要收费的，如果不去释放的话，会像这样。

每个月多花了很多钱 唉
