安装hexo和git

hexo需要node.js环境，请到官网下载
git为GitHub的上传工具，请到官网下载

安装hexo

在node.js和git安装完成后，安装hexo

这样我们就生成了最基础的博客，可以在浏览器中输入 https://localhost:4000 访问
