wireguard在各种设备上的使用方法

tunsafe官网：tunsafe

  wireguard官方也提供了很多的相关应用，也是很良心。

wireguard官网：wireguard

  • wireguard Windows

    tunsafe windows下载地址：tunsafe windows

    使用方法

1.官网下载，可选择最新版和旧版，支持win7和win10

2.下载好安装

3.编辑配置文件，点击edit config开始编辑，粘贴你的配置文件

4.编辑完成保存，点击 connect即可链接

  • wireguard mac

    1.安装官网工具 brew install wireguard-tools

    2.把配置文件复制保存成 wrieguard.conf（conf结尾）的文件

    3.command + shift + .找到/etc创建 wireguard文件夹

    4.将文件复制到wireguard

    5.关闭wg wg-quick down * *指原网卡的名字 一般为 wgopenwrt

    6.开启wg wg-quick up wireguard

tunsafe官方也提供了相应的mac os系统安装wireguard的方案：官网安装方式
